How to generate a PAT
  1. Go to github.com/settings/tokens (Settings → Developer settings → Personal access tokens → Tokens (classic))
  2. Click Generate new token (classic)
  3. Give it a name (e.g. "PR Dashboard")
  4. Set an expiration (e.g. 90 days)
  5. Check the repo scope
  6. Click Generate token
  7. Copy the token (starts with ghp_) and paste it above

Token is stored only for this browser tab (sessionStorage).
