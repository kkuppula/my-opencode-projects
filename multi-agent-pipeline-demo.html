Architecture Overview

Specialized agents, each with a single responsibility

🎯
Orchestrator
Routes requirements, selects pipeline mode, manages gates, assembles reports. Never writes code.
🔍
Discovery Agent
Read-only exploration. Finds files, understands patterns, produces implementation contract.
🛠️
Implementation Agent
Writes code following the discovery contract. Must comply with all guard rails.
🧪
Test Agent
Creates tests following existing conventions. Validates behavior changes.
✅
Verification Agent
Reviews diffs, runs checks, validates contract compliance. Generates PR description.
🧠
Knowledge Base
Patterns, pitfalls, and lessons learned. Gets smarter with every pipeline run.

Intelligent Mode Selection

Not every change needs the full pipeline — the right mode saves time and tokens

🚀
Full Feature
Discovery → Implementation → Testing → Verification
New features, complex changes
⚡
Quick Fix
Implementation → Verification
Typos, labels, config values
🚨
Hotfix
Discovery(lite) → Implementation → Verification
Critical production bugs
🔧
Refactor
Discovery → Implementation → Verification
Structural changes, no behavior change
